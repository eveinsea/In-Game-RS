In-Game Recommendation System

This page is used to simulate an embedded game recommendation system. When a player logs in or performs some specified operations, the system recommends new, attractive games to him/her from multiple perspectives. The system implements recommendations algorithms on popularity, content similarity, item based collaborative filtering and ALS collaborative filtering.
